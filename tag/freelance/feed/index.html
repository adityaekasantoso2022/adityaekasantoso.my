Web Development Timeline

Briefing

1 – 3 Hari

Kita dapat melakukan briefing sesuai dengan waktu yang sudah disepakati. Semua penjelasan tentang website yang akan dibuat bisa didiskusikan waktu ini.

Note : Bisa secara daring 

UI UX Design

1 – 7 Hari

Setelah saya memahami briefing yang diberikan dan mendapatkan beberapa referensi website, saya akan memulai proses desain tampilan antarmuka website. Di waktu ini kita akan berdiskusi tentang desain tampilan antarmuka yang saya desain.

Pembayaran DP sebesar 50% setelah desain antarmuka selesai.

Development

1 – 14 Hari

Setelah menemukan desain antarmuka yang cocok, kita akan memulai code development.

Handoff ke Client

1 – 3 Hari

Setelah proses development telah selesai, client dapat memeriksa website yang sudah hampir jadi dan mengkonfirmasi website yang kita kerjakan. 

Jika sudah terkonfirmasi, saya akan menaikkan website ke hosting dan mengkoneksikannya dengan domain (.com/.id/dll)

Saya akan memberikan user guide untuk client, agar client dapat menggunakan website dengan baik.

Final

Teknologi yang saya gunakan dalam pengembangan website ini adalah wordpress dan elementor, wordpress mempunyai keunggulan untuk kebutuhan post, read, delete, edit. Penggunaannya yang mudah bahkan untuk orang awam juga, jangan khawatir, jika masih kurang mengerti dengan user guide yang saya berikan, kamu bisa tanya langsung dengan saya via whatsapp/telegram.
