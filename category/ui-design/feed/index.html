Redesign Sycamore Landing Page

Organization Name

Sycamore Group

Description of the organization and its target audience

I don’t want copies of other websites or designs. Please be original, creative and work to the brief.

See Design

Brief

As the business is a group of businesses I need a good landing page which easily points visitors to the correct area of the website or service that they need.

I’d like a modern and slick website. Nothing too classic. The website needs to be quick and work on a mobile site as well as a desktop. I don’t want to use too much stock art as then I feel the site just looks too generic and unprofessional.

I would consider a single page layout where different different ‘pages’ are seen as you scroll down.

Naps Finance Website

Organization Name

Naps Finance (99design)

Description of the organization and its target audience

We’re launching a new cryptocurrency platform where we will be allowing other new cryptocurrencies to offer an initial public sale on our platform. Target audience is people who will buy new tokens.

Visit Website

Brief

We need a one pager to display information about our site (home page). Add generic info or lorem ipsum and i’ll edit it later when i have the info. Just make it so that there are 3 or 4 sections for adding conent.

The next page is a page where new tokens can be listed, bought etc.something like this, but put you’re own spin on it: ttps://app.poolz.finance/lobby

The last page needs to be more of a marketplace where people can buy and sell allocations, from other people (peer to peer), for the public sale. So if someone gets into the public sale, before the token is launched, they can then sell that allocation that they got to others, so we need a page for that. Just a generic marketplace type of design.
