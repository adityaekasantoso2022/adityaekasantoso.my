Program ini saya buat bertujuan untuk menguji pengaruh website untuk branding digital UMKM, saya sangat berharap pembuatan website untuk UMKM dapat membawa business growth yang signifikan.

Syarat dan Ketentuan

  1. Biaya Hosting dan Domain ditanggung sendiri
  2. Pengerjaan Website sesuai dengan nomor antrian
  3. Pembelian Hosting Wajib menggunakan Link Referral yang saya berikan
  4. Pengerjaan Website Tidak Dikenai biaya Sama Sekali
  5. Website digunakan untuk tujuan yang baik dan tidak merugikan orang lain
  6. Website akan mempunyai credit di paling bawah created by uiuxganysigit
  7. Website yang sudah jadi akan ditampilkan di website saya pribadi https://uiuxganysigit.com

Panduan Registrasi

  1. Follow dan berikan ulasan rekomendasi di Facebook Page UIUXganysigit dengan link https://www.facebook.com/uiuxganysigit/
  2. Kirim bukti follow dan ulasan ke chat Facebook Page UIUXganysigit.
  3. Jelaskan keperluan website untuk apa dengan rinci.
  4. Harap menunggu sebagai waiting list
  5. Sebagai bukti kamu telah terdaftar di waiting list, saya akan masukkan nama bisnismu di page waiting list (onprogress)
Winner

Pemenang Program Go Digital UMKM

  1. Pemenang harap melakukan pembelian hosting dengan link referral yang saya berikan di chat facebook setelah pemenang di umumkan.
  2. Harap melakukan pembelian hosting dengan link referral yang saya berikan.
  3. Saya tidak bertanggung jawab atas kesalahan kamu jika membeli hosting tidak dengan link referral yang saya berikan.
  4. Harap dipahami baik baik karena antrian akan hangus jika melakukan kesalahan di poin nomor 2.
  5. Jika pembeli tidak membeli hosting setelah (maksimal 1 bulan) pengumuman pemenang, maka antrian yang saya berikan akan hangus.Â 
  6. Untuk pertanyaan apapun bisa kamu tanyakan di whatsapp.
